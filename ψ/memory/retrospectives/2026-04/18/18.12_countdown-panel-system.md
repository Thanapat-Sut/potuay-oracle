## Session Summary

Added a unified countdown panel system to `BaseGameUI` so all games display countdown (3→2→1→GO!) on the projector screen alongside the existing LED Floor grid countdown. Included animated effects (elastic punch, rotation shake, shrink-fade, GO! color flash). Also restructured IceLand and TerritoryWars to play intro videos from the moment gameplay panel opens, waiting for video to finish before showing waiting tiles.

## Lessons Learned

- `SetAsLastSibling()` คือ standard fix สำหรับ UI z-order issues ใน Unity Canvas
- Fire-and-forget coroutine pattern (StartCoroutine without yield) ใช้ได้ดีกับ animation ที่ต้อง sync กับ timing ภายนอก
- Intro video → wait → setup tiles เป็น reusable pattern สำหรับเกมที่มี character intro
