**1. Wrong directory assumption** — ผมอ่าน SnakeWars ก่อนทั้งที่ plan บอก "SnakeWars" แต่ actual code อยู่ที่ SerpentWar directory ต่างหาก ควร scan working directories ที่ registered ใน environment ก่อน match กับ plan context

## Lessons Learned

- SerpentWar = refactored version ใน `Assets/Scripts/SerpentWar/` — ไม่ใช่ SnakeWars
